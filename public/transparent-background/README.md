# North transparent-background marks

Every file in this folder has a true transparent alpha background. There is no coloured square or canvas behind the artwork.

- `svg/`: scalable transparent vectors.
- `png/`: transparent PNGs exported at 1024 px wide.
- `png-hires/`: transparent PNGs exported at 2048 px wide.

Colour suffixes:

- `teal`: North deep charcoal-teal, `#193136`
- `white`: pure white, `#FFFFFF`
- `offwhite`: North warm off-white, `#F3EDE2`
- `black`: monochrome black, `#161A1B`

The app, Apple Touch and PWA icons live elsewhere because installable application icons require a complete background canvas.
