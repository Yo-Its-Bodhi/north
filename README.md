# North

Head North. Every day.

North is a daily companion for people trying to become a better version of themselves through fitness, food, recovery, learning, and one good decision at a time.

## Project status

**MVP 0.1 — Active build**

Current core product directions:

- Today — presence and direction
- Training — body
- Journey — memory
- Insights — understanding
- Nova — mind
- You — identity and personalisation (deferred until the core loop is proven)

The first runnable North experience now covers Today, workout preparation,
live set logging, pass-and-return, autosave/resume, and workout review.

Run it locally with:

```bash
npm install
npm run dev
```
