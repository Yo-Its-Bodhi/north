# 01 - North Ecosystem

Version: 0.0.1
Last updated: 2026-07-07
Author: Bodhi + GPT
Reason for change: Creates the full product ecosystem map before detailed screen design begins.

## Purpose

The North Ecosystem is the full map of the product.

It shows where every major feature belongs before screens are designed or code is written.

North should never become a random collection of features.

Every idea needs a home.

Every home needs a purpose.

## Ecosystem Philosophy

North is designed around experiences, not features.

Features only matter when they help the traveller:

- understand themselves
- make a better decision
- reduce friction
- build consistency
- reflect on progress
- keep heading North

## Core Product Pillars

North is built around five primary areas.

1. Today
2. Journey
3. Training
4. Nova
5. Profile

These are the main navigation pillars.

They should remain calm, clear and limited.

## Pillar 1: Today

Today is the home of North.

Not a dashboard.

The beginning of today's journey.

### Contains

- Nova greeting
- Daily check-in
- Today's focus
- Weather
- Quick start
- Workout
- Walk
- Run
- Bike
- Recovery
- Today's plan
- Journey snapshot
- This Day preview
- Today's milestone
- Manual quick logging

### Purpose

Help the traveller understand what matters today in less than five seconds.

## Pillar 2: Journey

## Secondary Systems

These systems support the five pillars.

### Health Integrations

- Apple Health
- Samsung Health / Health Connect
- Garmin
- Strava
- Fitbit
- WHOOP

Automatic data should make North easier, never necessary.

### Content System

- Exercise images
- Exercise videos
- Blueprint technique visuals
- Muscle diagrams
- Learning explanations
- Recovery guides
- Future recipes and cookbook

### Achievement System

North uses Milestones, not generic achievements.

Milestones should celebrate progress, effort, recovery, return, consistency and understanding.

### Memory System

Nova remembers what helps it care better.

Memory exists to create understanding, not surveillance.

### Notification System

North does not interrupt.

North invites.

Examples:

- Fancy a walk?
- Want to reset together?
- Your body might appreciate recovery today.

### Personalisation System

The dashboard belongs to the traveller.

North never rearranges it automatically.

Nova can suggest changes, but the user decides.

## Ecosystem Rule

If a feature does not fit clearly into Today, Journey, Training, Nova or Profile, it should go into the Parking Lot until its purpose is clearer.

## Founder Principle

North is not a collection of tools.

North is one connected experience designed to help a traveller keep heading in the right direction.
